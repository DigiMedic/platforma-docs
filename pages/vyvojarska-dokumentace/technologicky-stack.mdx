# Technologický stack

DigiMedic využívá moderní technologie pro zajištění výkonu, bezpečnosti a flexibility. Zde je přehled klíčových technologií použitých v projektu:

## Frontend

- **Framework**: Next.js
  - Server-side rendering, optimalizace výkonu, snadná integrace s React ekosystémem
- **UI knihovna**: Shadcn/UI
  - Předdefinované UI komponenty pro rychlý a přizpůsobitelný vývoj
- **JavaScript knihovna**: React
  - Efektivní tvorba uživatelského rozhraní

## Backend

Backend řešení není specifikováno v rámci boilerplate, ale platforma je připravena na integraci s různými backendy podle potřeb konkrétního projektu.

## Databáze

Flexibilní možnost výběru mezi:
- **Supabase**: PostgreSQL databáze s real-time funkcemi
- **Firebase**: NoSQL databáze s real-time synchronizací

## Autentizace

- **OAuth 2.0**: Bezpečné přihlašování a správa uživatelských účtů

## Nasazení

- **Vercel**: Optimalizováno pro Next.js aplikace, snadné nasazení a škálování
- **Docker**: Kontejnerizace pro konzistentní prostředí napříč různými systémy

## Integrace

Platforma je připravena na integraci s různými API a službami, včetně:
- Platebních bran
- SMS služeb
- Dalších zdravotnických systémů a API

## Mobilní vývoj

Technologie pro mobilní verzi bude určena později, ale platforma je navržena s ohledem na budoucí mobilní rozšíření.

Při vývoji dbejte na dodržování best practices pro každou z těchto technologií a pravidelně aktualizujte závislosti pro zajištění bezpečnosti a výkonu aplikace.
